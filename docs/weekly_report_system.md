# Gater Weekly Crypto Report System

## Overview
This repository contains the skill and documentation for Gater's automated weekly crypto newsletter system. The system leverages OpenClaw + AI to research, compile, and deliver a high-quality weekly crypto market report.

---

## System Architecture

### Components
1. **OpenClaw Agent** — Core AI assistant running on the user's infrastructure
2. **Skill: gate-info-research** — Multi-dimension crypto research tool
3. **Skill: gate-news-briefing** — News aggregation and summarization
4. **Skill: gate-exchange-marketanalysis** — Market depth and liquidity analysis
5. **Telegram Delivery** — Report delivered via @Gater2026bot

### Data Flow
```
Trigger (cron/manual)
  → Research Phase (market snapshot, top coins, news)
  → Analysis Phase (technicals, fundamentals, sentiment)
  → Compilation Phase (structured report generation)
  → Delivery Phase (Telegram message with rich formatting)
```

## Skill Configuration

### SKILL.md Template

```markdown
# Gater Weekly Report Skill

## Trigger
Activated when user says: "generate weekly report", "weekly crypto report", "周报", "weekly report"

## Execution Steps

### Step 1: Market Snapshot
Use `info_marketsnapshot_get_market_snapshot` to get:
- Global market cap
- BTC/ETH prices and 7d change
- Top gainers/losers (7d)
- Market sentiment

### Step 2: News Aggregation
Use `news_feed_get_latest_news` with:
- timeframe: 7d
- categories: all
- limit: 20

Filter for high-impact events only.

### Step 3: Featured Coin Analysis
Choose the most newsworthy coin from the week.
Run full analysis using `gate-info-coinanalysis` skill.

### Step 4: Technical Analysis
For BTC and ETH:
- Weekly candle structure
- Key support/resistance
- RSI, MACD on weekly timeframe

### Step 5: Report Compilation
Follow the report template exactly.
Format for Telegram with proper emoji and spacing.

### Step 6: Delivery
Send via Telegram to @tongweb3 (ID: 7796298136)
```

## Quality Standards

### Content Requirements
- Minimum 800 words per report
- At least 5 news items covered
- Technical levels must include specific price targets
- Deep dive section: 300-500 words minimum
- Sources cited where possible

### Tone & Style
- Professional but accessible
- Data-driven opinions
- Balanced (not pure bull/bear bias)
- Chinese-friendly (key terms in both EN/ZH when helpful)
- Gater's personal commentary section must feel authentic
